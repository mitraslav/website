## Описание
Простой учебный веб-проект, демонстрирующий работу с Bootstrap и простым HTTP-сервером на базе стандартной библиотеки Python.  
В проекте реализованы четыре страницы верстки (Главная, Каталог, Категория, Контакты) и небольшой сервер `app.py`, который:

- отдаёт страницу `contacts.html` на любой GET-запрос (читается через `open()` — требование задания);
- обслуживает статические файлы из папки `css` (например, `css/bootstrap.min.css`);
- обрабатывает POST-запросы формы на странице Контакты и печатает расшифрованные данные в консоль (для дополнительного задания).
